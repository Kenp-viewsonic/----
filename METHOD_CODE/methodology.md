# 毒性语义演化的认知记忆框架：面向FSCIL的统一视角 (Methodology Storyline)

## 核心理念 (TL;DR)
当前处理连续毒性评论识别（FSCIL）的方法多陷入“模块拼凑”的困境，缺乏对**“毒性语义如何演化”**这一语言学本质的深度建模。本研究摒弃了传统的静态正则化或完全参数隔离的思路，提出了一种**仿生认知记忆架构 (Cognitive Memory Architecture)**。

我们的统一范式认为：模型在面对层出不穷的变异黑话和新型仇恨言论时，其学习过程应类似于人类审核员的认知路径——**“表层特征感知 -> 锚定核心语义 -> 短期适应与长期沉淀 -> 未知边界防御”**。

基于此故事主线，我们将工程计划中的四大技术创新无缝融合为三个紧密耦合的理论模块：

---

## 模块一：毒性表达的异常感知与语义锚定 (感知层)
**对应工程实现：** `ToxicAwarePE` (特征) + `ToxicSemanticPrefix` (提示)
**解决的问题：** 模型如何对故意变异的黑话和层出不穷的新型侮辱保持“敏感”？

人类在审核文本时，往往先通过视觉符号（大写、奇怪符号）感受到“攻击性”，再将其与已知的毒性概念对齐。
为此，我们设计了**双重诱导机制**：
1. **结构异常感知 (ToxicAwarePE)：** 针对恶意变体（如 l33t 替换、规避空格、连续情绪符号）设计特化的位置编码。它在底层不仅捕获序列位置，更充当了“毒性表层异常的嗅探器”，为后续深层处理提供结构先验。这使得我们的方法**高度特化于毒性文本领域**，而非通用分类补丁。
2. **核心语义锚定 (Semantic Anchor Prefix)：** 面对新类别时，模型不能像无头苍蝇一样发散。我们通过在基座阶段提取的 K-means 质心作为前缀（Prefix），将输入文本强制拉近到已知的“毒性语义空间”。这是一种**基于原型的认知诱导**，确保后续的增量学习不会偏离仇恨和毒性的基本语义核心。

---

## 模块二：基于双分支架构的探索与沉淀 (记忆层) **【核心颠覆性创新】**
**对应工程实现：** `DualBranchLoRALayer` + `SemanticConsolidation`
**解决的问题：** 学了新词忘旧词？新词和旧词到底是什么关系？

现有方法要么用正则把参数“困住”（如EWC），要么为每个任务开辟独立的参数空间（如O-LoRA），导致知识碎片化。我们的认知论认为，毒性语言的演化包含两部分：**“易变的表层形式（新造黑话）”** 和 **“稳定的恶毒内核（仇恨本质）”**。

据此，我们提出了**动态语义沉淀范式**：
1. **可塑分支 (Plastic Branch - 短期工作记忆)：** 专门用于快速拥抱当前阶段的新词汇、新变体。它高度活跃，适应性强，致力于拟合当前的时代黑话。
2. **稳定分支 (Stable Branch - 长期核心记忆)：** 这是一个缓慢更新的参数容器，存储着跨越阶段的通用恶意逻辑。
3. **动态沉淀 (Semantic Consolidation)：** 当一个阶段结束时，我们并非简单粗暴地冻结参数。如果模型发现当前的“新黑话”其实只是对旧有仇恨内核的换皮（即对原有稳定参数干扰不大，$\delta_k < \tau$），就会将**可塑分支的经验合并（Add）到稳定分支中，实现“阅历沉淀”**，并清空可塑分支以迎接下一波网络新词；如果这是一种全新的毒性范式（干扰过大），则将其冻结为独立的历史快照。

这一机制赋予了模型**自适应进化的生命力**，彻底颠覆了传统CIL模型机械的灾难性遗忘防御策略。

## 模块三：开放环境下的认知边界防御 (防御层)
**对应工程实现：** `HierarchicalRejectionGate`
**解决的问题：** 当遇到连短期记忆都无法理解的东西时，模型如何自我保护？

认知框架的最后一环是“知道自己的无知”。在真实的开放网络环境中，模型必然会遇到完全不相关的文本（如广告代码）或脱离现有毒性框架的全新攻击形式。
我们在输出端构建了基于**表面异常与语义距离融合的层级拒识门控**。它不再仅依赖输出层 Softmax 的置信度，而是综合了输入层的 OOV 异常度与深层的距离原型距离（到预留锚点的距离）。
这样配置后，门控不再是一个孤立的“补丁”，而是整个认知模型闭环的“保安”：**当感知层和记忆层都无法将输入与已有毒性内核建立稳健联系时，执行拒识。**

## 实验与评估的故事呼应 (Evidence for Story)

为了向审稿人证明这不仅仅是一个漂亮的故事，而是真实发生在模型内部的机制，我们将通过以下指标和图表进行硬核自证：

1. **证明“沉淀”真实发生 (The Proof of Consolidation)：**
   - 记录训练日志中可塑分支与稳定分支激活权重的变化。
   - 展现：在遇到已知相似变体时，模型主要依赖 Stable 层；遇到全新类别时，Plastic 层被剧烈激活。
2. **证明“锚定”有效 (The Proof of Anchoring)：**
   - 引入 **CKA 语义稳定性评测 (Semantic Stability)**。证明在引入双分支和前缀后，模型第一阶段的 [CLS] 表征空间即使到了最后几轮依然坚如磐石，未发生严重漂移。
3. **证明“特化感知”的优势 (The Proof of Domain Specificity)：**
   - 核心特色指标：**变体召回率 (Variant Recall)**。我们的模型不仅要在标准测试集上好，更要在经过 l33t 替换、结构扭曲的对抗样本上大幅领先通用连续学习框架（如Seq-Finetune, Task-LoRA），证明我们在“毒性对抗”领域的不可替代性。
